# Vaylix Docs

This repository contains the documentation website for **Vaylix**.

The site is built with **Astro** and **Starlight** and is published to GitHub Pages at:

- `https://vaylix.github.io`

The main Vaylix product repository lives separately:

- `https://github.com/vaylix/vaylix`

## Local Development

Install dependencies and start the docs site:

```sh
npm install
npm run dev
```

Useful commands:

```sh
npm run build
npm run check
```

## Deployment

This site is deployed automatically with GitHub Actions when changes are pushed to `main`.
